# 动脉压力评估模块任务拆解

本文档用于把“二维压力分布动画 + 机器学习健康评估”拆成可执行任务。

## 1. 范围定义

- 输入：下位机上传点阵压力帧（文本协议或 Justfloat）。
- 展示：二维皮肤表面压力分布动画。
- 分析：波动次数、波动形态、一致性、重复性。
- 评估：机器学习输出健康评分与风险等级。

## 2. 里程碑

### M1 架构接入（低风险）

- 新增 `src/analytics` 模块骨架。
- 增加点阵适配器：channels -> HxW 矩阵。
- 新增分析管线（默认关闭，不影响现有流程）。
- 接入 `MainWindow.update_data` 的旁路调用。

验收标准：

- 现有功能可用，回归测试不退化。
- 关闭分析开关时行为与当前版本一致。

### M2 二维动画 MVP

- 新增二维压力矩阵缓存与更新节流。
- 输出实时压力矩阵给 UI 渲染层。
- 先提供结果数据接口，UI 可后续迭代。

验收标准：

- 连续输入时稳定输出矩阵。
- 缺帧/错帧可降级处理。

### M3 指标分析 MVP

- 实现基础指标：
  - 波动次数（bpm）
  - 幅值
  - 一致性分数
  - 重复性分数
- 结果进入统一 AnalysisResult。

验收标准：

- 指标数值随测试数据变化符合预期。

### M4 机器学习 MVP

- 特征提取器与模型运行器接入。
- 支持规则模式与外部模型模式。
- 输出健康评分、风险等级、解释字段。

验收标准：

- 在无模型文件时可规则降级。
- 在有模型文件时可输出预测结果。

## 3. 工作分解（WBS）

1. 数据契约与适配
- 定义输入帧校验。
- 实现 `pressure_grid_adapter.py`。

2. 分析管线
- 实现窗口缓存与节流。
- 实现结果缓存与重置逻辑。

3. 指标计算
- 时间域：峰值检测与 bpm 估算。
- 空间域：均值、方差、重心、面积。
- 重复性：窗口相关系数。

4. 机器学习
- 实现 `feature_extractor.py`。
- 实现 `model_runner.py`（规则 + 可选外部模型）。

5. 主链路集成
- 在数据消费后提交分析帧。
- 断连时重置分析状态。

6. 测试
- 适配器单测。
- 管线单测。
- 集成烟测（默认关闭不影响主流程）。

## 4. 风险与依赖

- 风险：实时链路被阻塞。
- 规避：分析旁路、节流、默认关闭。

- 风险：点位命名不规范导致无法映射。
- 规避：严格正则校验并输出告警。

- 依赖：下位机稳定发送全点位帧。

## 5. 优先级

- P0：M1 + M3（骨架 + 指标）
- P1：M4（ML）
- P2：动画 UI 增强
