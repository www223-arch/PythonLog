# Python上位机 - 数据采集与显示

一个支持多种数据源、实时波形显示和CSV数据保存的Python上位机软件。

## ✨ 功能特性

### 数据采集
- ✅ **UDP数据源**：支持UDP网络数据接收
- ✅ **数据校验头**：支持自定义数据校验头，过滤无效数据
- ✅ **时间戳支持**：使用发送方发送的时间戳作为X轴坐标
- ✅ **自动通道检测**：根据接收到的数据自动创建通道
- ✅ **多通道支持**：支持最多32个通道，每个通道自动分配颜色

### 数据显示
- ✅ **实时波形显示**：多通道实时波形显示
- ✅ **频域分析**：支持FFT频域分析，自动检测峰值
- ✅ **通道颜色自定义**：支持右键菜单自定义通道颜色
- ✅ **通道显示/隐藏**：支持单独控制每个通道的显示状态

### 数据交互
- ✅ **缩放功能**：鼠标滚轮缩放波形
- ✅ **平移功能**：鼠标拖拽平移波形
- ✅ **鼠标吸附**：鼠标移动时显示最近数据点坐标
- ✅ **双击标记**：双击标记数据点，计算两点差值
- ✅ **频域标记**：双击频域曲线标记频率点

### 数据管理
- ✅ **CSV数据保存**：自动保存接收到的数据为CSV格式（Excel可打开）
- ✅ **缓存区管理**：可设置缓存区大小，控制数据点数量
- ✅ **数据限制**：支持限制数据点数，避免内存溢出
- ✅ **清空数据**：一键清空所有通道数据

### 显示控制
- ✅ **暂停/继续**：支持暂停和继续显示（数据继续接收和保存）
- ✅ **自动保存**：连接后自动开始保存数据
- ✅ **实时状态**：显示连接状态、数据计数、保存文件信息

### 可扩展架构
- ✅ **模块化设计**：数据源、可视化、数据处理分离
- ✅ **插件式架构**：支持添加新的数据源和数据处理方法

## 🚀 快速开始

### 1. 启动上位机

```bash
python src/main.py
```

### 2. 发送测试数据

打开另一个终端，发送测试数据到上位机：

**基本用法**：
```bash
# 查看帮助信息
python tools/udp_sender.py --help
```

**发送正弦波数据**：
```bash
# 发送3通道正弦波，50Hz采样率，持续10秒
python tools/udp_sender.py --type sine --channels 3 --frequency 1.0 --duration 10 --rate 50

# 发送3通道正弦波，使用自定义通道名称
python tools/udp_sender.py --type sine --channels 3 --frequency 1.0 --duration 10 --names 电压 电流 温度
```

**发送随机数据**：
```bash
# 发送3通道随机数据，持续10秒
python tools/udp_sender.py --type random --channels 3 --duration 10
# 发送3通道随机数据，使用自定义通道名称
python tools/udp_sender.py --type random --channels 3 --duration 10 --names 电压 电流 温度
```

**参数说明**：
- `--host`: 目标主机地址（默认：127.0.0.1）
- `--port`: 目标端口（默认：8888）
- `--type`: 数据类型，`sine`（正弦波）或 `random`（随机数据）
- `--channels`: 通道数量（默认：3）
- `--frequency`: 正弦波频率（Hz，默认：1.0，仅正弦波有效）
- `--amplitude`: 正弦波振幅（默认：1.0，仅正弦波有效）
- `--duration`: 发送持续时间（秒，默认：10.0）
- `--rate`: 采样率（Hz，默认：50，建议50-100Hz）
- `--names`: 自定义通道名称，用空格分隔（例如：`--names 电压 电流 温度`）
- `--header`: 数据校验头（默认：DATA）

**示例**：
```bash
# 示例1：发送正弦波数据（默认配置）
python tools/udp_sender.py --type sine

# 示例2：发送随机数据（默认配置）
python tools/udp_sender.py --type random

# 示例3：发送正弦波数据，自定义参数
python tools/udp_sender.py --type sine --channels 2 --frequency 2.0 --amplitude 1.5 --duration 20 --rate 100

# 示例4：发送随机数据，自定义通道名称
python tools/udp_sender.py --type random --channels 3 --duration 15 --names 电压 电流 温度 --header DATA

# 示例5：发送到指定主机和端口
python tools/udp_sender.py --type sine --host 192.168.1.100 --port 9999 --duration 5
```

## 📖 使用说明

### 上位机界面

#### 1. UDP配置
- **主机地址**：默认 `0.0.0.0`（监听所有接口）
- **端口**：默认 `8888`
- **数据校验头**：默认 `DATA`，用于过滤无效数据
- **连接/断开**：点击"连接"按钮开始接收数据，点击"断开"停止接收
- **暂停/继续**：暂停时保持连接，图像不打印，保存继续,快捷键--空格

#### 2. 通道配置
- **自动检测**：程序会自动检测接收到的通道
- **通道列表**：显示所有检测到的通道名称
- **通道颜色**：每个通道自动分配颜色（支持32种颜色）
- **缓存区大小**：可设置缓存区大小，控制数据点数量（默认1000）
- **数据限制**：可限制数据点数，避免内存溢出（默认启用）
- **清空通道**：一键清空所有通道数据

#### 3. 通道管理
- **设置通道颜色**：右键点击通道名称标签，选择"设置通道颜色"，然后选择要修改的通道
- **通道显示/隐藏**：在波形显示区域点击图例，可单独控制每个通道的显示状态
- **颜色预览**：通道颜色与曲线颜色一致，便于识别

#### 4. 数据保存
- **自动保存**：连接后自动开始保存数据
- **保存目录**：可自定义保存目录（默认 `data/`）
- **文件格式**：CSV格式，Excel可直接打开
- **文件名格式**：`data_YYYYMMDD_HHMMSS.csv`
- **手动控制**：可手动开始/停止保存

#### 5. 时域波形显示
- **缩放**：鼠标滚轮缩放波形
- **平移**：鼠标拖拽平移波形
- **鼠标吸附**：鼠标移动时显示最近数据点坐标
- **双击标记**：双击标记数据点（最多2个点，自动计算差值）
- **暂停显示**：点击"暂停"按钮暂停显示
- **清空数据**：点击"清空所有通道"按钮清空数据

#### 6. 频域分析
- **切换视图**：点击"频域分析"标签切换到频域视图
- **选择通道**：在下拉菜单中选择要分析的通道
- **执行分析**：点击"执行FFT分析"按钮进行频域分析
- **峰值检测**：自动检测并显示前5个峰值
- **主频标记**：自动标记主频点
- **双击标记**：双击频域曲线标记频率点
- **清除标记**：点击"清除标记"按钮清除用户添加的标记点
- **采样率设置**：可设置采样率（默认100Hz）
- **频率信息**：显示主频、幅值、频率范围、峰值等信息

#### 7. 状态显示
- **连接状态**：显示当前连接状态（已连接/未连接）
- **数据计数**：显示已接收的数据数量
- **保存文件**：显示当前保存的文件名

### UDP数据格式

**文本格式（推荐）**：
```
数据校验头,时间戳,通道一=数值,通道二=数值,通道三=数值
```

例如，3通道数据：
```
DATA,123.456,通道一=1.234,通道二=2.567,通道三=0.890
```

**说明**：
- **数据校验头**：第一个字段必须是数据校验头（字符串，默认'DATA'）
- **时间戳**：第二个字段必须是时间戳（浮点数）
- **通道名称**：`=` 前面的名称将作为通道名
- **通道数量**：根据实际数据自动检测
- **数据校验**：程序会验证数据校验头，不匹配的数据将被丢弃
- **无需手动添加**：程序会自动根据通道名创建对应的显示通道

**Python发送示例**：
```python
import socket
import time

sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

# 构建数据
header = 'DATA'  # 数据校验头，必须与上位机设置一致
timestamp = time.time()
data = f"{header},{timestamp},通道一=1.234,通道二=2.567,通道三=0.890"

# 发送数据
sock.sendto(data.encode('utf-8'), ('127.0.0.1', 8888))
```

**CSV数据格式**：
```
时间戳,通道一,通道二,通道三
123.456,1.234,2.567,0.890
123.457,1.235,2.568,0.891
...
```

- 文件保存在 `data/` 目录
- 文件名格式：`data_YYYYMMDD_HHMMSS.csv`
- Excel可直接打开

## 🔧 开发指南

### 添加新的数据源

1. 继承 `DataSource` 基类：
```python
from src.data_sources.base import DataSource

class MyDataSource(DataSource):
    def connect(self) -> bool:
        # 实现连接逻辑
        pass
    
    def read_data(self):
        # 实现读取数据逻辑
        pass
    
    def disconnect(self) -> None:
        # 实现断开连接逻辑
        pass
```

2. 在主程序中使用：
```python
from src.data_sources.manager import DataSourceManager

manager = DataSourceManager()
source = MyDataSource()
manager.set_source(source)
```

### 自定义数据处理

在 `src/main.py` 的 `update_data()` 方法中添加自定义处理逻辑：
```python
def update_data(self):
    data = self.data_source_manager.read_data()
    if data is not None:
        # 添加自定义数据处理
        processed_data = your_processing_function(data)
        
        # 更新波形显示
        self.waveform_widget.update_channels(processed_data)
```

## ❓ 常见问题

### Q1: 为什么看不到曲线？

**A:** 可能的原因：
1. 数据源未连接：检查"状态"中的连接状态
2. 通道被隐藏：检查波形显示区域的图例，确保通道未被隐藏
3. 数据格式错误：检查数据格式是否符合要求（数据校验头,时间戳,通道名=数值）
4. 数据校验头不匹配：检查"数据校验头"设置是否与发送方一致

### Q2: 如何修改通道颜色？

**A:** 右键点击"检测到通道: xxx"标签，选择"设置通道颜色"，然后选择要修改的通道。

### Q3: 如何隐藏某个通道？

**A:** 在波形显示区域点击对应的图例，可单独控制每个通道的显示状态。

### Q4: 如何进行频域分析？

**A:** 切换到"频域分析"标签，选择要分析的通道，点击"执行FFT分析"按钮。

### Q5: 数据保存在哪里？

**A:** 默认保存在 `data/` 目录，文件名格式为 `data_YYYYMMDD_HHMMSS.csv`。可以在"数据保存"区域自定义保存目录。

### Q6: 如何调整显示的数据点数量？

**A:** 在"缓存区大小"输入框中输入新的数值，点击"应用"按钮。同时确保"限制数据点数"复选框已选中。

### Q7: 暂停时数据还在接收吗？

**A:** 是的，暂停时数据继续接收和保存，只是波形显示暂停。点击"继续"按钮后，波形从最新接收的数据开始显示。

### Q8: 支持多少个通道？

**A:** 最多支持32个通道，每个通道自动分配不同的颜色。

### Q9: 如何标记数据点？

**A:** 双击波形曲线上的任意位置，可以标记数据点。最多可以标记2个点，系统会自动计算两点之间的差值。

### Q10: 频域分析需要多少数据点？

**A:** 至少需要10个数据点才能进行频域分析。数据点越多，频域分析越准确。
